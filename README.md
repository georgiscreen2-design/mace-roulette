# mace-roulette
Made a simple plugin for mace roulette

Step 1 
Download the .jar file

Optiona steps if you dont have a minecraft server:
Step 2 
Download a paper server for any 1.21 version(Recommended for 1.21.10-1.21.11)
https://papermc.io/downloads/paper

Step 3 
Create a folder, named it.(Something like Mace Roulette)

Step 4
Put the .jar file for the server in the folder you created and click on it until use see eula.txt(that may take a time or clicks)

Step 5
Open the eula.txt and on the spot where you see false change it to true(Read the EULA if you want to know what you agree on) and save the file

Step 6 
Click the .jar file for the server. The server will start and a window will open on your screen 

Step 7
Close/Stop the server by clicking on the cross, go to the plugins folder and put the Mace Roulette .jar file. Open the server again by clicking on the server .jar file

Step 8 
Join the server through this ip 127.0.0.1(thats your local ip) make yourelf op. You can make that by going to the console and writhing op {your_minecraft_username} and use /mace reward

Step 9 
Put some items in the gui that will open when typing the command /mace reward and click ESC, the rewards will save and you can play now

Step 10
Go on the servers with your friends and type /mace new. Enjoy!
